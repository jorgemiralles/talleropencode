# OpenCode Agent Instructions

## Overview
This is a static HTML project - a single CV/resume webpage. There is no build system, framework, or package manager.

## Architecture
- Single file: `cv.html` - edit directly
- No npm, yarn, or any package manager
- No build step required

## Workflow
- **No tests, linting, or typechecking** - none are configured
- **No dev server** - open `cv.html` directly in a browser to view changes
- **No CI/CD** - no automated workflows exist

## Common Mistakes to Avoid
- Do not run `npm install`, `yarn`, or any package manager commands
- Do not search for build scripts, Makefiles, or configuration files
- Do not attempt to run test suites or linters
